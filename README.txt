RAG-ассистент для генерации откликов на вакансии

описание:  
    RAG-ассистент помогает формировать персонализированные отклики на вакансии с использованием NLP и LLM.  
    Он анализирует текст вакансии и резюме соискателя, находит релевантные примеры из базы (few-shot) и строит prompt для LLM,  
    чтобы сгенерировать связный, профессиональный отклик, максимально соответствующий требованиям вакансии.

как формируется prompt:  
1. ассистент загружает резюме соискателя и извлекает ключевые навыки и опыт.  
2. на основе текста вакансии ищет 2–3 похожие вакансии с откликами в базе (few-shot примеры).  
3. объединяет текст вакансии, выбранные примеры откликов и ключевые данные резюме в один prompt.  
4. передаёт prompt в LLM для генерации готового отклика, который затем выводится пользователю.
